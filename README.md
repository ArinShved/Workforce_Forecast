# ИТР: Анализ и прогнозирование рынка труда

Аналитическая система для исследования и прогнозирования баланса спроса и предложения на рынке инженерно-технических кадров (ИТР) на примере Курской области.

Проект включает:
- Обработку и анализ данных о приеме, выпуске и трудоустройстве.
- Расчет коэффициентов сохранности контингента и кадрового дефицита.
- Сценарное прогнозирование (пессимистичный, базовый, оптимистичный сценарии).
- Интерактивную визуализацию в виде десктопного приложения на Qt.

## Стек технологий

- **Язык:** Python 3.9+
- **База данных:** PostgreSQL
- **Основные библиотеки:** Pandas, NumPy, SQLAlchemy, Psycopg2
- **Графический интерфейс:** PySide6 (Qt)
- **Визуализация:** Matplotlib
- **Конфигурация:** ConfigParser

## Возможности системы

### Вкладка «Приём / Выпуск»
- Отображает динамику приема и выпуска по специальностям. 
- Строит прогноз выпуска ИТР-кадров на ближайшие годы.
- Показывает сводку по сохранности контингента.

### Вкладка «Дефицит кадров»
- Сравнивает количество вакансий, выпускников и безработных.
- Выводит список профессий с наибольшим дефицитом и профицитом.

### Вкладка «Рынок по должностям»
- Анализирует соотношение вакансий и безработных по конкретным должностям.
-Предоставляет информацию о средней, минимальной и максимальной зарплате.

### Вкладка «Сценарный прогноз»
- Прогнозирует баланс спроса и предложения на 5 лет вперед.
- Позволяет сравнить три сценария развития рынка.
- Строит тепловую карту дефицита и графики по выбранной специальности.
